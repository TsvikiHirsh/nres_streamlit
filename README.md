# nres_streamlit

Streamlit server for nres: Simple yet powerful package for fitting of neutron resonances

## Installation


From source:
```bash
git clone https://github.com/TsvikiHirsh/nres_streamlit
cd nres_streamlit
python -m pip install .
```
